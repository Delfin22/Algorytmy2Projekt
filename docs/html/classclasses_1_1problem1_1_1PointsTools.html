Calculates the total length of the convex hull formed by a list of points.

The length of each wall is the Euclidean distance between two neighbouring points in the list. The total length of the convex hull is the sum of the lengths of all the walls.

This function generates random edges between points in a list.

The function shuffles the input list of points and then adds an edge between each consecutive pair of points in the shuffled list. It also adds an edge between the last point in the shuffled list and the first point to form a closed loop.

Parameters
pointsA list of Point objects representing the points in a graph.
Returns
A list of Point objects in the order they were connected.
